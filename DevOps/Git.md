# Git Commands

```yaml
git init #initializes a local git repo
git clone http://.....git #creates a local copy of the remote repo
git add . #adds all new & changed files to staging area
git commit -m "commit message" #commits changes
git branch #lists all the branches with the corrent branch indicated as an asterik
git branch test #creates a new branch named test
git checkout -b test #creates a new branch called test and switches to it

```
